## ライセンスと改変について

このリポジトリは、[hugo-book](https://github.com/alex-shpak/hugo-book) のデザインを流用して再実装したものです。

- オリジナルの hugo-book のライセンスを継承しています（流用・派生部分）。
- 本リポジトリには、オリジナル実装からの改変が含まれます。

このリポジトリで利用しているライセンス本文は [LICENSE](./LICENSE) に記載しています（[hugo-book/LICENSE](https://github.com/alex-shpak/hugo-book/blob/main/LICENSE) から転載）。

オリジナルの詳細は [hugo-book/README.md](https://github.com/alex-shpak/hugo-book/blob/main/README.md) を参照してください。

英語版は [README.md](./README.md) を参照してください。
